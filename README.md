# Security & its Applications
Contains cryptographic assignments

##### Requirements
- Python 3

##### How to run the code
- Test for DES:
`
python3 main.py des [msg] [key]
`
- Test for AES:
`
python3 main.py aes [msg] [key]
`
- Test for Vernam:
`
python3 main.py vernam [msg] [key]
`
- Test for Caesar:
`
python3 main.py caesar [msg] [shift]
`
